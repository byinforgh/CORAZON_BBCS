CORAZON Assistant
Ask about revenue pace, which clients to chase, how to word a follow-up or a referral ask, or anything else about running the business. It sees your live revenue, follow-up and client-health numbers when you ask. Requires an ANTHROPIC_API_KEY set in Apps Script Script Properties — see the Setup tab for the one-time step.
Business Identity
This only controls how the dashboard displays itself — it does not touch your existing SETUP tab (product catalog, tax codes, etc).
AI Assistant Setup (one-time)
1. Open the spreadsheet → Extensions → Apps Script.
2. Project Settings (gear icon) → Script Properties → Add script property.
3. Property: ANTHROPIC_API_KEY — Value: your Anthropic API key. Save.
4. Reload this dashboard. The AI Assistant tab will start working immediately, no redeploy needed.

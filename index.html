AI Assistant Setup (one-time)
1. Open the spreadsheet → Extensions → Apps Script.
2. Project Settings (gear icon) → Script Properties → Add script property.
3. Property: ANTHROPIC_API_KEY — Value: your Anthropic API key. Save.
4. Reload this dashboard. The AI Assistant tab will start working immediately, no redeploy needed.
